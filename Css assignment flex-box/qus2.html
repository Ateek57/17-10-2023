Role: This property defines how extra space within a flex container is distributed along the main axis. It affects the alignment of items in that direction and can take values like flex-start, flex-end, center, space-between, and space-around.

align-item

Role: It determines how flex items are aligned within the flex container along the cross-axis. Common values include flex-start, flex-end, center, baseline, and stretch

Flex-direction

Role: This property determines the direction in which flex items are placed within a flex container. It can be set to values like row, row-reverse, column, or column-reverse

Flex-wrap

Role: Flex items, by default, all try to fit on one line. flex-wrap controls whether items wrap onto a new line when they don't fit within the container. Values include nowrap, wrap, and wrap-reverse
